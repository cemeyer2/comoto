  • What is CoMoTo?

    The Collaboration Modeling Toolkit is a dynamic web application that allows university course staff to run automated collaboration analysis on source code files using a variety of detection engines, mechanisms, and techniques. Access to the CoMoTo instance is currently restricted to course staff in the Department of Computer Science.
